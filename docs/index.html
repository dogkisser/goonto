Can I use images from my hard drive?

In the config, paste the path to your images into the source_path field. Note that backslashes (\) should be replaced with either forward slashes (/), or with two backslashes (\\).

Can I change what E621 images I see?

In the config, place tags in the source_tags array. You can add multiple tags per string, for example: "source_tags": ["female solo", "male duo -female"], will result in you seeing both solo females and gay images.
